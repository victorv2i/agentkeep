# Contributing

Thanks for helping Agentkeep.

## Development

```bash
pnpm install
pnpm test
pnpm typecheck
pnpm --filter @agentkeep/web build
```

Use Node 22 or newer.

## Expectations

- Write tests first for logic changes.
- Keep changes small and aligned with existing style.
- Keep copy concise and factual.
- Open a fork and send a pull request.

Do not include secrets, private vault data, or generated screenshots with real personal data.
